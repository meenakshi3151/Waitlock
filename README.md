# Student Entry Management System  
A full-stack MERN application designed to manage student entry/exit records, authentication, and automated email alerts for late students.

---

## **Project Overview**
This project provides a web interface for student registration, login, tracking IN/OUT records, and sending automated late-entry emails. It includes a React frontend and a Node.js/Express backend with MongoDB as the database.

---

## **Features**
### **Frontend (React)**
- User Signup & Login  
- Student dashboard  
- Students IN list  
- Students OUT list  
- Toast notifications  
- Navigation bar, footer, and basic pages  

### **Backend (Node.js + Express)**
- User authentication routes  
- Email sending route  
- OTP verification route  
- Entry/Exit logging route  
- CORS configured for React  
- MongoDB connection
